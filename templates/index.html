URL Shortener - API Documentation

This is a simple URL Shortener in Golang! This service allows you to shorten URLs and retrieve stored shortcuts. Below are the available endpoints:

/new

Type: POST

This endpoint allows you to add a new URL to be shortened. It returns the shortcut for the given URL.

Request Body: The body of the request should be of type x-www-form-urlencoded and must include a key "url" with the destination URL as its value.

/list

Type: GET

This endpoint returns a list of all registered shortcuts, allowing you to see all stored URLs.

/s/<shortcut>

Type: GET

This endpoint redirects you to the URL registered under the specified shortcut.
